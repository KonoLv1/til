# Windowsコマンドプロンプト用

## cd

現在のディレクトリを変更する

```
cd desktop
```
デスクトップにディレクトリを移動する

## ipconfig

ネットワーク設定を確認する

```
ipconfig
```

## ping

ネットワーク疎通を確認する
```
ping
```

**TTL（Time To Live - 生存時間）**<br>
ループ防止目的のために設定されてある、パケットが有効である時間で、レイヤー３のネットワーク機器を一つ通過するごとに値が1つ減っていく。<br>
送信元に表示されるTTL値 = 基礎値 - 到達までのネットワーク機器数
